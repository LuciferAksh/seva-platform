# Frontend Starter

Recommended stack:

- React
- Vite
- Tailwind CSS
- Leaflet

## Primary User Flows

### Field Worker

- submit voice note
- submit survey image
- submit quick text report
- see upload confirmation

### Coordinator

- monitor live need map
- filter by urgency and category
- review AI-extracted records
- assign volunteers
- track completion

## Suggested Screens

- landing or login
- field submission page
- coordinator dashboard
- volunteer detail drawer
- need detail modal

## UX Notes

- keep field-worker UI extremely simple
- optimize for mobile first
- make coordinator dashboard feel like mission control
- show match explanation clearly
